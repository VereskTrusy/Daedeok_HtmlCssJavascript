1. 요소의 접근
 1) 자바스크립트 문법으로 요소 접근 방법
    document.getElementById('요소명');
    '요소명'의 요소들 모두 선택됨, #아이디명과 .클래스명을 '요소명'자리에 써줘서 접근도 가능하다

 2) jQuery 문법으로 요소 접근 방법 
    $('요소명')
    '요소명'의 요소들 모두 선택됨, #아이디명과 .클래스명을 '요소명'자리에 써줘서 접근도 가능하다
   
2. HTML요소 렌더링 후 함수 호출
 1) 요소 렌더링 이전 함수호출하여 요소 접근 시 오류 발생한다
         그러므로 요소가 모두 렌더링 후 함수호출하고 요소에 접근 하는데,
         몇가지 방법을 알아보자
 
 2) 자바스크립트 문법으로 요소 렌더링 후 함수 호출 방법
   window.onload = function(){ 
       // 로드 후 실행할 로직
   }
 
 3) jQuery 문법으로 요소 렌더링 후 함수 호출 방법 1
    $(document).ready( function(){ 
        // 로드 후 실행할 로직
    })
    
 4) jQuery 문법으로 요소 렌더링 후 함수 호출 방법 2
    $(function(){
        // 로드 후 실행할 로직
    });
