jQuery를 사용하여 이벤트 선언 시 여러 형태로 선언 할 수 있다
1. 이벤트의 선언
  1) 자바스크립트로 이벤트 선언 하기 방법 1
     document.getElementById('#btn1').onclick = function(){
     	// 실행 할 로직
     }
  2) 자바스크립트로 이벤트 선언 하기 방법 2
  	 document.getElementById('#btn1').addEventListener('click', test()); // 클릭 이벤트 시 함수 호출 할당
  	 document.getElementById('#btn1').removeEventListener('click', test()); // 클릭 이벤트 시 함수 제거
  	 
  	 // 함수는 따로 작성
  	 function test(){
  	 	// 실행 할 로직
  	 }
  	 
  3) jQuery에서 이벤트 함수 선언 하기 - 사용 o
     $('#btn1').on('click', function(){
     	// 실행 할 로직
     });
     
  4) jQuery에서 이벤트 함수 선언 하기 - 사용 x
     $('#btn1').bind('click', function(){
     	// 실행 할 로직
     });
  
  5) jQuery에서 이벤트 함수 선언 하기 - 사용 x
     $('#btn1').click(function(){
     	// 실행 할 로직
     });
