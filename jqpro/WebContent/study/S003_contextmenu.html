contextmenu를 사용하여 마우스 우클릭을 막아보자
설정한 요소 위에서 마우스 우클릭 동작 시 발생하는 이벤트가 contextmenu 이벤트이다.
현재 html문서 모든 곳에서 마우스 우클릭을 막기위해 요소를 html로 지정하였고
함수의 매개변수로 이벤트 객체 e를 받아 기본  우클릭 이벤트에 대한 기본동작을 막았다.

* e.preventDefault() : 주로 링크클릭, 폼제출, 우클릭메뉴표시 등의 기본동작을 막을때 사용

// 문서 로드 후 실행되는 함수
$(function(){
	$('html').on('contextmenu', function(e){ // 범위 html
		e.preventDefault(); // 기본 동작 취소
		alert("공공자료 보호를 위해 마우스 우클릭이 제한되어 있습니다.");
	});
});
